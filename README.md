# Products Dashboard

Files included:

- index.html
- style.css
- dashboard.js

Usage:

1. Mở `index.html` trong trình duyệt (double-click hoặc serve bằng static server).
2. Trang sẽ tự động gọi `getAll()` tải dữ liệu từ https://api.escuelajs.co/api/v1/products

Chức năng đã thực hiện:

- Tải tất cả sản phẩm bằng hàm `getAll()`.
- Hiển thị ảnh đầy đủ, không cắt.
- Tìm kiếm theo `title` (live `input` và `change`).
- Phân trang với lựa chọn 5/10/20 mỗi trang.
- Nút sắp xếp theo `title` và `price` (tăng/giảm).
- Ẩn cột `description` mặc định, hiển thị khi hover vào dòng.

Ghi chú về ảnh chụp màn hình (yêu cầu của bài):

- Vui lòng mở trang trong trình duyệt và chụp ảnh màn hình khi bạn thử mỗi tính năng: tìm kiếm, phân trang, sắp xếp, hover để hiển thị mô tả.
